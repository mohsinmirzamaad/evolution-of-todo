# Project Overview: Evolution of Todo

## What We Are Building
A Todo application that evolves across 5 phases — from a simple
Python console app to a cloud-native AI-powered distributed system.

## Current Phase
Phase I — In-Memory Python Console App

## Features for Phase I
1. Add Task — create a new todo item
2. View Tasks — list all tasks with status
3. Update Task — modify an existing task
4. Delete Task — remove a task by ID
5. Mark Complete — toggle task completion status

## Tech Stack for Phase I
- Python 3.13+
- uv for package management
- In-memory storage (no database)
- Claude Code + Spec-Kit Plus
